Javascript Integration Test

Teste a integração de seus Adlayer Widgets

Bookmarklet

  1. Apenas arraste Adlayer Integration Test para sua barra de favoritos
  2. Vá para a página integrada
  3. Clique no bookmarklet na barra de favoritos

Script

  1. Insira o seguinte script no html de uma página integrada com adlayer: 
    <script src="http://integration.adlayerjavascriptsdk.com/main.js"></script>
  2. Atualize a página (Command + R or F5)
Veja um exemplo online

Rodando os testes

  1. Depois de executar o bookmarket e carrregar a interface de teste
  2. Clique em "Rodar"
  3. Confira os relatórios, se algum teste não passar (vermelho), algo está errado com sua integração *
* Se todos os testes estão verder, então você fez um bom trabalho e sua integração está funcionando perfeitamente

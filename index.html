Javascript Integration Test

Test your Adlayer widget integration

Bookmarklet

  1. Just drag the Adlayer Integration Test to your bookmarks bar
  2. Go to the integrated page
  3. Click on bookmarklet item

Script

  1. Insert following script in the html of integrated page: 
    <script src="http://integration.adlayerjavascriptsdk.com/main.js"></script>
  2. Refresh the page (Command + R or F5)
See a live example

Running Tests

  1. Select the type of integration you wanna test
  2. Click on "Run" button inside selected tab
  3. Check the reports, if any test do not pass (red), there's something wrong with our integration *'
* If all tests are green, you did a good job and your integration is working perfectly
